# Purpose

Ensure all exhibit content is written in a clear, interpretive curatorial voice, not as trivia or process explanation.

## What Done Looks Like

- All new and revised exhibit text interprets significance, not just facts.
- No process-explainer or directory-style language on public surfaces.
- Captions, room intros, and object labels are concise and interpretive.

## Constraints / Non-Goals

- Does not cover technical documentation or sprint docs.
- Does not require rewriting legacy content unless touched for other reasons.
