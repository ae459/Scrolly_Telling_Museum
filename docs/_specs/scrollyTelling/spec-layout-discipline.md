# Purpose

Reinforce Bauhaus-inspired layout discipline and visual hierarchy across all exhibit surfaces.

## What Done Looks Like

- All pages use consistent grid, spacing, and alignment.
- Titles, captions, and cards follow standardized visual rules.
- No overflow, misalignment, or ad hoc styling.

## Constraints / Non-Goals

- Does not specify color palette or typography changes unless needed for alignment.
- Does not address media asset selection.
